const int yask::yask_numa_local = -1

Allocate grids on local NUMA node.

This is used in yk_solution::set_default_numa_preferred and yk_grid::set_numa_preferred. In Python, specify as yask_kernel.cvar.yask_numa_local.

const int yask::yask_numa_interleave = -2

Allocate grids across all available NUMA nodes.

This is used in yk_solution::set_default_numa_preferred and yk_grid::set_numa_preferred. In Python, specify as yask_kernel.cvar.yask_numa_interleave.

const int yask::yask_numa_none = -9

Do not specify any NUMA binding.

This is used in yk_solution::set_default_numa_preferred and yk_grid::set_numa_preferred. In Python, specify as yask_kernel.cvar.yask_numa_none.
